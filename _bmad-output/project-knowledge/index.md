### Getting Started
1. Open `PiTrainer/PiTrainer.xcodeproj` in Xcode 15 or later.
2. Select a simulator (e.g., iPhone 16) or a physical device.
3. Press **Cmd+R** to run the app.
4. Press **Cmd+U** to run the unit tests.
